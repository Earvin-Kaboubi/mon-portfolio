Site Web

Conception et développement d'un site web complet et responsive, réalisé dans le cadre du BTS SIO option SLAM.

HTML5 CSS3 JavaScript MySQL
Accueil Réalisations Site Web

Présentation du projet

Dans le cadre de ma formation en BTS SIO option SLAM, j'ai réalisé un site web dynamique complet. Ce projet m'a permis de mettre en pratique l'ensemble des compétences acquises en développement front-end et back-end.

Le site propose une interface moderne et responsive, adaptée à tous types d'écrans. Il inclut une base de données MySQL gérée via PHP pour la persistance des données, ainsi qu'un espace d'administration sécurisé.

Les fonctionnalités principales incluent un système d'authentification, une gestion de contenu dynamique, et un formulaire de contact fonctionnel avec validation côté client et serveur.
