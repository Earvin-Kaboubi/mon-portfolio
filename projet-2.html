Projet 2ème Année

Application web complète développée en deuxième année de BTS SIO SLAM, intégrant les meilleures pratiques du développement moderne.

PHP JavaScript MySQL REST API Bootstrap
Accueil Réalisations Projet 2ème Année

Présentation du projet

En deuxième année de BTS SIO SLAM, j'ai réalisé une application web complète en PHP avec une architecture MVC structurée. Ce projet représente l'aboutissement de ma formation et mobilise l'ensemble des compétences acquises.

L'application propose une gestion complète avec authentification, rôles utilisateurs, tableaux de bord dynamiques et une API REST pour communiquer avec des services externes. L'interface front-end utilise JavaScript moderne (Fetch API, DOM dynamique).

Le projet a été développé en suivant une méthodologie agile avec des sprints hebdomadaires, ce qui m'a permis d'itérer rapidement et de livrer un produit fonctionnel à chaque étape.

Technologies

PHP 8 JavaScript MySQL REST API Bootstrap Git

Fonctionnalités principales

Authentification

Système de connexion sécurisé avec gestion des sessions, hachage des mots de passe et rôles utilisateurs.

CRUD Complet

Gestion complète des données avec validation des formulaires, gestion des erreurs et retours utilisateur.

API REST

Endpoints PHP pour exposer les données en JSON, consommés depuis le front-end via JavaScript Fetch API.

Bilan du projet

Ce projet de deuxième année m'a permis de consolider mes compétences en développement full-stack et d'acquérir de nouvelles pratiques professionnelles : architecture MVC, travail en équipe avec Git, conception d'API REST.

La rigueur apportée à la sécurité (protection XSS, injections SQL, gestion des sessions) reflète les bonnes pratiques enseignées en BTS SIO. Ce projet constitue la base de mes compétences pour intégrer le monde professionnel.
