Projet 1ère Année

Application de gestion développée en première année de BTS SIO SLAM. Découverte du développement applicatif et des bases de données relationnelles.

Java SQL MySQL
Accueil Réalisations Projet 1ère Année

Présentation du projet

En première année de BTS SIO SLAM, j'ai développé une application de gestion en Java .Ce projet constitue mon premier développement applicatif complet dans le cadre de ma formation.

L'application permet la gestion d'un stock ou d'une liste d'éléments avec les opérations CRUD connectées à une base de données MySQL locale.

Ce projet m'a permis d'acquérir les bases du développement orienté objet en Java, la manipulation de bases de données via JDBC, et la création d'interfaces utilisateur avec Swing.

Technologies

Java JDBC MySQL

Détails

  • Année 2024–2025
  • ~4 semaines
  • Projet individuel
  • 1ère année BTS SIO

Compétences mobilisées

  • Programmation Java OOP
  • Connexion JDBC/MySQL
  • Modélisation UML

Étapes du projet

1

Modélisation

Réalisation du diagramme de classes UML et conception du schéma de la base de données relationnelle.

2

Base de données

Création des tables MySQL, rédaction des requêtes SQL et configuration de la connexion JDBC.

3

Interface Java

Développement de l'interface graphique, implémentation des formulaires et des tableaux de données.

Bilan et apprentissages

Ce premier projet a été une expérience fondatrice. J'ai appris à structurer un projet de développement, à travailler avec les principes de la programmation orientée objet et à interfacer une application avec une base de données.

Les difficultés rencontrées — notamment la gestion des exceptions JDBC et la mise en place de l'architecture MVC — m'ont permis de progresser rapidement et de mieux appréhender les projets plus complexes de deuxième année.
